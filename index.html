1. Lisensi dan Penggunaan

1.1. Lisensi Terbatas: Pengembang memberikan Anda lisensi yang terbatas, non-eksklusif, tidak dapat dialihkan, dan dapat ditarik untuk menggunakan program ini hanya untuk keperluan pribadi dan non-komersial Anda.

1.2. Pembatasan Penggunaan: Anda tidak diizinkan untuk:

  • Menjual, menyewakan, meminjamkan, atau mendistribusikan program ini tanpa persetujuan tertulis.
  • Membalikkan rekayasa (reverse-engineer), mendekompilasi, atau mencoba menemukan kode sumber program.
  • Menggunakan program ini untuk tujuan ilegal atau yang tidak sah.
  • Mengklaim program yang telah dimodifikasi sebagai karya asli Anda.

1.3. Fungsionalitas Notifikasi Telegram: Anda memahami dan setuju bahwa:

  • Notifikasi akan dikirimkan ke akun Telegram yang Anda tautkan.
  • Notifikasi dapat berisi data spesifik seperti nomor/email akun dan NIK.
  • Anda bertanggung jawab penuh atas keamanan akun Telegram Anda.

2. Keterbatasan dan Penafian

2.1. Program "Apa Adanya": Program ini disediakan "apa adanya" dan "sebagaimana tersedia" tanpa jaminan dalam bentuk apa pun. Pengembang secara tegas menafikan semua jaminan, termasuk kelayakan jual dan kesesuaian untuk tujuan tertentu.

2.2. Penafian Tanggung Jawab: Pengembang tidak bertanggung jawab atas kerugian atau kerusakan apa pun yang timbul dari penggunaan program ini. Ini termasuk kehilangan data, kerugian finansial, atau kerusakan akibat bug. Pengembang tidak menjamin program akan berfungsi tanpa gangguan.

3. Kekayaan Intelektual

3.1. Hak Cipta dan Kepemilikan: Program ini, termasuk semua kode sumber, desain, dan dokumentasi, adalah milik pengembang dan dilindungi oleh undang-undang hak cipta. Perjanjian ini tidak memberi Anda hak kepemilikan apa pun.

3.2. Merek Dagang: Semua merek dagang, logo, dan nama layanan adalah merek dagang dari pengembang. Anda tidak diizinkan untuk menggunakannya tanpa persetujuan tertulis.

4. Privasi Pengguna

4.1. Pengumpulan Data: Pengembang tidak mengumpulkan, menyimpan, atau mentransmisikan data pribadi Anda. Semua data diproses secara lokal di perangkat Anda dan dikirim langsung ke API Telegram tanpa melalui server pengembang.

4.2. Persetujuan Telegram: Anda bertanggung jawab untuk memberikan persetujuan yang diperlukan kepada Telegram untuk menerima notifikasi dari program ini.

5. Pengakhiran

5.1. Pengakhiran oleh Pengembang: Pengembang berhak mengakhiri lisensi Anda kapan saja jika Anda melanggar S&K ini. Setelah pengakhiran, Anda harus segera berhenti menggunakan dan menghapus semua salinan program.

5.2. Pengakhiran oleh Pengguna: Anda dapat mengakhiri perjanjian ini kapan saja dengan berhenti menggunakan dan menghapus program.

6. Hukum yang Berlaku
